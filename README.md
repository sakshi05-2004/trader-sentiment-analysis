# Trader Performance vs Market Sentiment Analysis

This project analyzes how Bitcoin market sentiment (Fear vs Greed) influences trader behavior and profitability using Hyperliquid trading data.

## Key Metrics
- Daily PnL
- Win Rate
- Trade Frequency
- Average Trade Size

## Key Insights
- Fear markets show highest profitability
- Traders trade more frequently during Fear
- Win rates increase during volatile periods

## Strategy Ideas
- Increase trading activity during Fear markets
- Reduce leverage during Extreme Greed
- Avoid overtrading in Neutral markets

## How to Run

Install dependencies:

pip install -r requirements.txt

Run the notebook:

jupyter notebook trader_sentiment_analysis.ipynb
